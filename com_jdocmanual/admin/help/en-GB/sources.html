Sources

The Sources page contains a list of Manuals and indicates their published states. Edit a Manual to change its published state.

Unpublish Deleted

When an article is deleted from a Markdown source (or renamed) its HTML conversion is not deleted from the database. The Unpublish Deleted button is a utility that unpublishes deleted articles so they do not appear in search results.

Important: Rerun the Smart Search Indexer after unpublishing articles.

Build

Select a Language for a Manual to rebuild the HTML content and Menus after any change. The first time the build process is run it may take 5 to 10 minutes to complete. Subsequent runs are much quicker because only changed source files are rebuilt.

Pull

This column should be present if you need to pull changes to the Markdown files from the GitHub repositories. Otherwise it can be omitted in the Jdocmanual configuration options. When select, it does a git pull for the selected language and manual.
